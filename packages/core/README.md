# 🏗️ AEM Core Library (@repo/core)

**Single Source of Truth | Shared Constants | Business Logic**

ศูนย์กลางข้อมูลและกฎเกณฑ์ (Business Logic) สำหรับทั้ง Monorepo

## 📂 ส่วนประกอบสำคัญ (Core Components)

- `constants/area-nodes`: ข้อมูล 77 จังหวัดไทยแบบ Structured Data (P-SEO Hub)
- `constants/site-config`: การตั้งค่าแบรนด์, ลิงก์สังคมออนไลน์, และสโลแกน (Single Truth)
- `lib/area-processor`: ฟังก์ชันสำหรับการคำนวณและประมวลผลข้อมูลภูมิภาค
- `types`: ส่วนกลางสำหรับการกำหนด Type ที่ใช้ร่วมกันทั่วทั้งระบบ
