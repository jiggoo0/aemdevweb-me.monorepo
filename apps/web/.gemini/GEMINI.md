# 🏢 AGENCY HUB - THE WEB NODE
**สถานะ:** โหนดศูนย์กลางเอเจนซี่ (Agency Hub Node) ของ UNLINK THAILAND Ecosystem
**การสืบทอด:** โหนดนี้สืบทอดกฎและมาตรฐานทั้งหมดจาก `home::GEMINI.md`

## 🎯 บทบาทหน้าที่ (Role)
- เป็นศูนย์กลางการนำเสนอบริการดิจิทัล (Digital Service Showcase) และเครื่องมือสร้าง SEO Template
- ขับเคลื่อน **AEMDEVWEB Central Engine** ผ่านการทำ SEO Dominance ใน 77 จังหวัด

## ⚖️ ข้อกำหนดความตั้งใจ (Intentionality Mandate)
- **ห้ามแก้ไขโดยพลการ:** AI ต้องไม่ทำการ "แก้ไข", "ปรับปรุงให้สวยงาม" (beautify), หรือเปลี่ยนแปลง metadata, การสะกดคำที่ตั้งใจ (เช่น "นายเอ็มซ่าส์มาก"), หรือคอมเมนต์พิเศษ (เช่น //DEV) ที่ผู้ใช้ระบุไว้
- **การรักษาเจตจำนง:** ทุกสิ่งที่ปรากฏในโค้ดหรือเอกสารที่ดูเหมือนเป็นข้อผิดพลาด ให้ถือว่าเป็น "เจตจำนงสุดท้ายของผู้ใช้" (Final Expression of User Intent) และต้องได้รับการรักษาไว้อย่างเคร่งครัด

## 🖋️ ลายเซ็นและเครื่องหมาย (Signatures)
- **Static Text Signatures:** ลายเซ็น "9mza", "เจ้าป่า", และ "นายเอ็มซ่าส์มาก" คือเครื่องหมายระบุตัวตนที่คงที่ (Immutable Markers)
- **ข้อห้าม:** ห้ามนำลายเซ็นเหล่านี้ไปตีความเป็นคำสั่งในการออกแบบ (Design Directives), ปรับเปลี่ยนสไตล์ UI, หรือเปลี่ยนแปลงโครงสร้างสถาปัตยกรรมโดยเด็ดขาด

## 🛠️ LOCAL MANDATE
1. **Execution Mode:** Technical Executor
2. **Next.js 16+ Standard:** โหนดนี้ใช้ Next.js 16+ ซึ่งมี Breaking Changes จากข้อมูลการเทรนเดิม (เช่น Async dynamic APIs) AI ต้องอ้างอิงเอกสารในโปรเจกต์และปฏิบัติตามคำเตือน Deprecation อย่างเคร่งครัด
3. **Scope Locking:** AI ต้องโฟกัสเฉพาะแผนงานใน `.gemini/plans/CURRENT_PLAN.md` ของโปรเจกต์นี้เท่านั้น และ **ห้าม** ตีความหรือนำแผนงานเชิงกลยุทธ์จาก Node อื่นหรือ Root มาใช้ เว้นแต่จะได้รับคำสั่งโดยตรง
4. **AI Context Isolation:** ไฟล์และโฟลเดอร์ที่เกี่ยวข้องกับการทำงานของ AI ทั้งหมด (เช่น plans, state, logs, reports) ต้องถูกจัดเก็บไว้ภายใน `.gemini/` เท่านั้น เพื่อรักษา Project Root ให้สะอาดและเป็นไปตามมาตรฐานการทำงานของโปรเจกต์
5. **Project Structure Update:** AI ต้องอัปเดตไฟล์ `PROJECT_STRUCTURE.txt` ไว้ภายใน `.gemini/` ทุกครั้งที่มีการเปลี่ยนแปลงโครงสร้างหลัก เพื่อให้บริบทการทำงานล่าสุดมีความแม่นยำสูงสุด
6. **Status Reporting:** เมื่อมีการเรียกใช้คำสั่งสรุปสถานะ (เช่น `/test status`) ให้ AI สรุปความคืบหน้าของงาน Agency และสถานะของ SEO Engine เป็นภาษาไทยที่กระชับและเป็นทางการ
