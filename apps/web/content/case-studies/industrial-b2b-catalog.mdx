# 🏭 Industrial B2B UX/UI Design

## 🌟 Design Stylings: The "Technical Hub"

ดีไซน์ที่เน้นประสิทธิภาพและความแม่นยำ (Technical Accuracy) โดยใช้โครงสร้างแบบ Grid ที่แข็งแรงและสีที่เน้นจุดสนใจ (Action-oriented colors)

### 🖱️ UX Features ที่ใช้งานได้จริง:

- **Spec-Comparison Table:** ตารางแสดงคุณสมบัติทางเทคนิคที่สามารถเลื่อนดูได้ (Horizontal scroll) บนมือถือ
- **One-Click PDF Specs:** ปุ่มดาวน์โหลดใบสเปคสินค้าแบบรวดเร็วโดยไม่ต้องเปลี่ยนหน้า
- **Bulk Quote Request:** ระบบรวบรวมรายการสินค้าเพื่อขอใบเสนอราคาหลายรายการพร้อมกันในคลิกเดียว
- **High-speed Search:** ช่องค้นหาที่แสดงผลลัพธ์ทันที (Type-ahead) ช่วยให้ค้นหา Part number ได้รวดเร็ว
