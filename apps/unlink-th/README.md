# 🛡️ UNLINK Thailand Registry (apps/unlink-th)

**Trust Verification | Integrity Auditing | Specialized Utilities**

แอปพลิเคชันสำหรับตรวจสอบความน่าเชื่อถือและการยืนยันอัตลักษณ์ดิจิทัลภายใต้เครือข่าย UNLINK

## 🚀 คุณสมบัติหลัก (Features)

- **Verification Portal:** ระบบตรวจสอบ Site ID และความถูกต้องของใบรับรองสถาบัน
- **Audit Reports:** แสดงผลคะแนนความน่าเชื่อถือ (Trust Score) และประสิทธิภาพด้านความปลอดภัย
- **Admin Registry:** จัดการข้อมูล Leads และรายงานการตรวจสอบระบบ

## 🏗️ โครงสร้างภายใน (Structure)

- `app/verify`: ระบบตรวจสอบ Site ID แบบ Real-time
- `app/audit`: รายงานการตรวจสอบเชิงเทคนิคระดับลึก
